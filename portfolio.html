Portfólio

Aqui estão alguns projetos que desenvolvi durante meus estudos de programação. Cada projeto representa um passo na minha jornada de aprendizado.

  • Carteira Bancária

    Este sistema simula um ambiente bancário simplificado, com funcionalidades de criação de contas, transações financeiras, controle de saldo e investimentos.
    Feito em Java.

    Ver no GitHub

  • Sistema de Catálogo e Pedidos

    Este sistema é uma arquitetura de microsserviços para um sistema de catálogo e pedidos, usando tecnologias Spring como Spring Boot, Spring Cloud, Eureka e API Gateway para criar uma solução autônoma e resiliente.
    Feito em Java.

    Ver no GitHub

  • Dashboard de salários na área de dados

    Este projeto é um dashboard interativo para análise de salários na área de dados. Possui gráficos interativos com Plotly mostrando salários médios, distribuição salarial, tipos de trabalho e salários por país.
    Feito em Python.

    Ver no GitHub

  • Quiz para programadores

    Este projeto foi meu primeiro contato com o desenvolvimento web. Trata-se de um quiz interativo com perguntas sobre conceitos básicos e tecnologias de front-end, servindo como uma demonstração das minhas habilidades iniciais.
    Feito com HTML e CSS

    Ver no GitHub

  • Jogo da Forca

    Um jogo da forca que roda no terminal. O jogador deve adivinhar a palavra com base em uma dica, tendo apenas 6 tentativas. Conta com palavras sortidas de várias categorias, cada uma com dica.
    Feito em Java
